# 2026-04-20 更新日志

## 改进内容

### 1. 新增 `material_uploader.py` — 资料上传服务
**文件**: `ppe_demo/services/material_uploader.py`

功能：
- 从 `materials.json` 读取资料元数据，将本地文件上传到飞书知识库对应课程节点下
- 按 `course_name` 自动匹配 `wiki_structure.json` 中的知识库节点
- 上传流程：上传文件到飞书云空间 → 将文件添加到知识库节点

支持的命令行参数：
```
python -m ppe_demo.services.material_uploader [--mode incremental|full] [--dry-run] [--verbose] [--delay N]
```

- `--mode incremental`（默认）：增量模式，跳过已上传的资料
- `--mode full`：全量模式，重新上传所有资料
- `--dry-run`：试运行，只检查不实际上传
- `--delay N`：上传间隔秒数（默认1.0）

上传记录保存在 `data/upload_records.json`，支持断点续传。

### 2. 增强 `feishu_service.py` — 错误处理和重试机制
**文件**: `ppe_demo/services/feishu_service.py`

改进：
- 新增 `@api_retry` 装饰器，支持指数退避重试
- 对 `get_tenant_access_token`、`create_wiki_node`、`move_doc_to_wiki`、`upload_file` 四个关键方法添加重试
- 自动处理 `TimeoutException` 和 `ConnectError`，最多重试3次
- 添加 `logging` 模块支持，日志输出函数名和重试次数
