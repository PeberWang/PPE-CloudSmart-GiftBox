# 多维表格字段设计

## 概述

按学年创建4个多维表格（大一/大二/大三/大四），每个表格列出该学年的专业课。每行对应一门课程，字段如下：

## 字段方案

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 课程名称 | 文本 (1) | 课程全称，如"中国经济概论" |
| 授课老师 | 文本 (1) | 老师姓名，如"龚关老师" |
| 开课学期 | 单选 (3) | 秋季/春季，选项：大一上、大一下、大二上、大二下、大三上、大三下、大四上、大四下 |
| 课程类型 | 单选 (3) | 必修/选修 |
| 考试形式 | 单选 (3) | 闭卷/开卷/论文/其他 |
| 学习指南 | URL (15) | 飞书知识库中文档链接 |
| 最后更新 | 日期 (5) | 最后一次更新资料的时间 |

## 飞书字段类型对照

| 类型编号 | 飞书UI类型 | 用途 |
|----------|-----------|------|
| 1 | Text | 文本字段 |
| 2 | Number | 数字字段 |
| 3 | SingleSelect | 单选字段 |
| 5 | DateTime | 日期字段 |
| 15 | Url | 链接字段 |

## 与现有资料表/心得表的关系

学年多维表格是**面向新生的入口导航**，而资料表和心得表是**后台管理表**。两者通过"课程名称"关联：

```
学年多维表格（新生视角）     后台管理表（运营视角）
┌──────────────────┐      ┌──────────────────┐
│ 课程名称         │───→  │ 课程（资料表）    │
│ 学习指南（链接）  │      │ 资料类型/文件链接  │
│ 资料数量         │      │ 审核状态          │
└──────────────────┘      └──────────────────┘
```
